
Build a complete production-ready Flutter mobile app called GaamRide.
This is a village-level ride + logistics platform for rural Gujarat, India.

---

## APP OVERVIEW

GaamRide has TWO modules in one app:
1. GaamRide — Person transport (like Rapido for villages)
2. GaamHaul — Vehicle/tempo booking for farmers (like Porter for villages)

THREE user roles:
- Customer — books rides or vehicles
- Gaam Saathi — provides bike/auto rides  
- Haul Saathi — owns tempo/truck, accepts haul bookings

Same login for all roles. Role detected from Firestore after login.
One person can be both Gaam Saathi AND Haul Saathi.

Service area: 9 villages in Mahuva taluka, Surat district, Gujarat.
Villages: Anaval, Kos, Tarkani, Angaldhara, Dholikuva, 
          Lakhavadi, Unai, Doldha, Kamboya

## CORE REAL-TIME TRACKING IMPLEMENTATION

This is the most important feature. Implement exactly as follows:

### Saathi location update (Saathi app, every 5-10 seconds):
```dart
// When ride is active:
Timer.periodic(Duration(seconds: 5), (timer) {
  Position pos = await Geolocator.getCurrentPosition();
  
  // Update ride document with current Saathi location
  await FirebaseFirestore.instance
    .collection('rides')
    .doc(rideId)
    .update({
      'saathiLat': pos.latitude,
      'saathiLng': pos.longitude,
      'saathiLastUpdate': FieldValue.serverTimestamp(),
    });
    
  // Also update saathis collection for discovery
  await FirebaseFirestore.instance
    .collection('saathis')
    .doc(saathiId)
    .update({
      'position': GeoFirePoint(pos.latitude, pos.longitude).data,
      'lastSeen': FieldValue.serverTimestamp(),
    });
});
```

### Customer real-time tracking (Customer app):
```dart
// Listen to ride document for Saathi location changes:
FirebaseFirestore.instance
  .collection('rides')
  .doc(rideId)
  .snapshots()
  .listen((snapshot) {
    final data = snapshot.data()!;
    final lat = data['saathiLat'];
    final lng = data['saathiLng'];
    final status = data['status'];
    
    // Animate marker to new position
    _animateSaathiMarker(LatLng(lat, lng));
    
    // Handle status changes
    if (status == 'accepted') showArrivingUI();
    if (status == 'started') showRideStartedUI();
    if (status == 'completed') navigateToCompleteScreen();
  });
```

### Smooth marker animation:
```dart
void _animateSaathiMarker(LatLng newPosition) {
  // Animate from current position to new position
  // Use a Timer + linear interpolation for smooth movement
  final startPos = _saathiMarkerPosition;
  final steps = 20;
  int step = 0;
  
  Timer.periodic(Duration(milliseconds: 50), (timer) {
    step++;
    if (step >= steps) { timer.cancel(); return; }
    
    final lat = startPos.latitude + 
      (newPosition.latitude - startPos.latitude) * step / steps;
    final lng = startPos.longitude + 
      (newPosition.longitude - startPos.longitude) * step / steps;
      
    setState(() {
      _saathiMarkerPosition = LatLng(lat, lng);
      _markers = _buildMarkers(); // rebuild with new position
    });
  });
}
```

## RIDE REQUEST → ACCEPT FLOW (FCM Notifications)

### When customer books a ride:
1. Create rides/{rideId} document with status: 'searching'
2. Find nearest available Saathi (already fetched on home screen)
3. Send FCM push notification to that Saathi's fcmToken:

—

Title: "નવી સવારી વિનંતી! 🛵" Body: "${customerVillage} → ${destinationVillage} | ${distance} કિમી" Data: { rideId: rideId, type: 'ride_request’ }
4. Customer sees "Searching..." animation

### When Saathi receives FCM:
- App is foreground → show bottom sheet popup with ride details
- App is background → notification tray, tap opens app to ride request

### When Saathi accepts:
1. Update rides/{rideId}: 
   status: 'accepted', saathiId, saathiName, saathiPhone
2. Send FCM to customer: "સાથી મળ્યો! ${saathiName} આવી રહ્યા છે"
3. Navigate Saathi to SaathiRideScreen
4. Customer RideRequestScreen auto-navigates to RideTrackingScreen

## FARE CALCULATION

### GaamRide:
```dart
double calculateFare(double distanceKm) {
  const double baseFare = 20.0;     // ₹20 base
  const double perKmRate = 8.0;     // ₹8 per km
  const double minimumFare = 30.0;
  
  double fare = baseFare + (distanceKm * perKmRate);
  return fare < minimumFare ? minimumFare : fare;
}
// Example: 3 km → ₹20 + ₹24 = ₹44
```

### GaamHaul:
```dart
double calculateHaulFare(int ratePerHour, String duration) {
  final hours = {
    '1h': 1.0, '2h': 2.0, 
    'half_day': 4.0, 'full_day': 8.0
  };
  double baseFare = ratePerHour * hours[duration]!;
  double appCommission = 75.0; // flat ₹75 booking fee
  return appCommission; // customer pays owner directly, app charges commission
}
```
